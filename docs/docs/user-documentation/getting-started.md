# Getting Started

This guide will help you get started with the Eretz Realty Admin System, from logging in to navigating the interface.

## System Requirements

### Browser Requirements

The Eretz Realty Admin System works best with modern web browsers:

- **Google Chrome** (recommended): Version 90 or higher
- **Mozilla Firefox**: Version 88 or higher
- **Microsoft Edge**: Version 90 or higher
- **Safari**: Version 14 or higher

### Screen Resolution

- **Minimum**: 1024 x 768 pixels
- **Recommended**: 1920 x 1080 pixels or higher

The system is fully responsive and works on tablets and mobile devices, though a desktop experience is recommended for extensive data entry.

## Accessing the System

To access the Eretz Realty Admin System:

1. Open your web browser
2. Navigate to your organization's Eretz Realty URL (provided by your administrator)
3. You will be directed to the login page

## Logging In

### First-Time Login

If this is your first time logging in:

1. You will receive login credentials from your system administrator, including:
   - Your email address (username)
   - A temporary password

2. On the login page, enter:
   - **Email/Username**: Your assigned email address
   - **Password**: Your temporary password

3. Click the **Login** button

4. After successful login, you will be redirected to the Dashboard (Cycle Manager)

### Standard Login Process

For subsequent logins:

1. Navigate to the login page
2. Enter your email address and password
3. Click **Login**
4. You will be redirected to the page you were last viewing, or to the Dashboard if this is your first login

### Login Issues

If you experience login problems:

- **Incorrect Credentials**: Verify that you're entering the correct email and password (passwords are case-sensitive)
- **Account Disabled**: If your account has been deactivated, contact your administrator
- **Forgot Password**: Contact your administrator to reset your password

## Understanding the Interface

After logging in, you'll see the main application interface with the following elements:

### Navigation Sidebar

Located on the left side of the screen, the sidebar provides access to all main sections:

- **Dashboard Icon**: Returns to the Cycle Manager (home page)
- **Cycle Manager**: Main dashboard with cycle overview
- **All Listings**: Complete property listing management
- **Schedule**: Email campaign configuration and schedule settings
- **Users** (Admin only): User account management

#### Collapsible Sidebar

The sidebar can be collapsed to provide more screen space:

1. Click the **menu icon** (three horizontal lines) at the top of the sidebar
2. The sidebar will collapse to show only icons
3. Hover over icons to see tooltips with section names
4. Click the menu icon again to expand the sidebar

The sidebar state is saved automatically, so it will remain collapsed or expanded based on your preference.

### Top Bar

The top bar displays:

- **Page Title**: The name of the current page
- **Page Description**: A brief description of the current section
- **Action Buttons**: Context-specific actions (e.g., "Add Listing" button on the Listings page)

### User Information

At the bottom of the sidebar, you'll find:

- **Your Email**: Your logged-in account email
- **Logout Button**: Click to sign out of the system

### Main Content Area

The central area displays the content for the current page, including:

- Data tables
- Forms
- Dashboard widgets
- Settings panels

## Navigation

### Moving Between Sections

To navigate to different sections:

1. Click any item in the sidebar navigation
2. The page will load and display the relevant content
3. The active section is highlighted in the sidebar

### Quick Tips

- **Responsive Design**: The interface adapts to your screen size
- **Mobile Access**: On mobile devices, tap the menu icon to show/hide the sidebar
- **Keyboard Navigation**: Use Tab to move between form fields and interactive elements
- **Tooltips**: Hover over buttons and icons to see helpful tooltips

## User Roles

The system has two user roles with different permissions:

### Standard User

Standard users can:
- View and manage listings
- Access the Cycle Manager dashboard
- Configure email schedules
- Manage email recipients
- Send test emails
- Manage lookup tables (property types, conditions, etc.)

### Administrator

Administrators have all standard user permissions, plus:
- Create new user accounts
- Reset user passwords
- Activate/deactivate user accounts
- Delete user accounts
- Full system configuration access

Your role determines which features and sections you can access. If you need additional permissions, contact your system administrator.
